# PixelFinder 🔎

PixelFinder is a **modern image search web application** that allows users to explore high-quality photos using the **Unsplash API**. Users can search for images, browse trending photos, preview images in fullscreen, download them, and navigate results using pagination.

The application features a **clean modern UI**, **dark mode**, and a **responsive layout** that works across devices.

---

## 🚀 Features

* 🔍 Search images using **Unsplash API**
* 🌄 View **Trending Images**
* 📄 **Pagination** for browsing results
* 🖼 **Fullscreen Image Preview (Modal)**
* ⬇ **Download Images**
* 🌙 **Dark Mode Toggle**
* 🎨 Modern **Responsive UI**
* 📱 Mobile-friendly design

---

## 🛠 Tech Stack
